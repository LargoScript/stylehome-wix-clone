# Style Homes - New Project

Новий проект сайту Style Homes, створюється з нуля (ну майже)

## Структура проекту

```
stylehome_new/
├── css/
│   └── style.css
├── js/
│   ├── libs/
│   │   ├── anime.min.js
│   │   ├── aos.css
│   │   └── aos.js
│   └── main.js
├── docs/
│   ├── ANIMEJS_DOCUMENTATION.md
│   ├── AOS_DOCUMENTATION.md
│   └── README.md
├── img/
├── video/
├── index.html
└── README.md
```

## Технології

- **HTML5** - структура сайту
- **CSS3** - стилізація з використанням сучасних технік
- **JavaScript (Vanilla)** - інтерактивність та анімації
- **AOS (Animate On Scroll)** - анімації при прокрутці
- **Anime.js** - складні анімації та ефекти
